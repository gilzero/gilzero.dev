Artisan Styleguide

Access to "/artisan-styleguide" to preview styleguide (it requires "administer themes" permission).

Use module provided SDC as a reference "artisan_styleguide:artisan-styleguide-sdc-model", read notes on its files to check details about how to define properly.

Also note styleguide builder is an overridable service just in case you would like to use an adjusted builder.
